## Role & Usage

Gemini CLI is an additional AI coding assistant for this repository, acting as a fallback for:
- long-context planning
- repository review
- research-heavy tasks
- documentation
- terminal-based coding assistance

It must follow the shared instructions in [AGENTS.md](./AGENTS.md) and the product/technical documentation in `docs/`.

## Operational Mode

Gemini CLI is configured to use **Plan Mode** by default. Always verify plans before execution.

## Prohibited Modes

- YOLO mode is strictly prohibited in this repository.
